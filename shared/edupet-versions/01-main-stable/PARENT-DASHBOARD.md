# EduPet Parent Dashboard (PR 2)

Path: `shared/edupet/dashboard/index.html`

## Features
- Current pet snapshot (mood/stage/stats/totals)
- Recent event history table
- Per-app totals table
- Manual event entry
- Export JSON
- Import JSON
- Reset state

## Usage
1. Open dashboard page.
2. Review state and event trends.
3. Enter manual events for testing.
4. Export backup JSON or import from a previous export.

## Privacy model
- Data is stored only in browser `localStorage`.
- No backend transmission in this version.
