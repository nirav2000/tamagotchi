# EduPet versions

This folder separates the main EduPet variants into versioned paths so different branches and experiments can be preserved without overwriting each other.

## Structure

- `01-main-stable/` — current `main` branch EduPet
- `02-ui-pass/` — UI-focused variant from `assistant-ui-pass`
- `03-lifecycle-pass/` — lifecycle-focused variant from `assistant-lifecycle`

## Purpose

Use these folders to compare the different EduPet directions side by side before deciding what becomes the official active implementation.
